# Python - Variable Annotations

## In this project, I learnt :

Type annotations in Python 3

How you can use type annotations to specify function signatures and variable types

Duck typing

How to validate your code with mypy

## Requirements

Allowed editors: vi, vim, emacs

All my files will are interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.9)

All my files end with a new line

The first line of all my files are #!/usr/bin/env python3

Your code use the pycodestyle style (version 2.5.)

All my files are executable

The length of my files are tested using wc

All my modules have a documentation (python3 -c 'print(__import__("my_module").__doc__)')

All my classes have a documentation (python3 -c 'print(__import__("my_module").MyClass.__doc__)')

All my functions (inside and outside a class) have a documentation (python3 -c 'print(__import__("my_module").my_function.__doc__)' and python3 -c 'print(__import__("my_module").MyClass.my_function.__doc__)')
