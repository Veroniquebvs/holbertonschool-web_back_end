# Python - Async Comprehension

## In this project, I learnt :

How to write an asynchronous generator

How to use async comprehensions

How to type-annotate generators

## Requirements :

Allowed editors: vi, vim, emacs

All my files are interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.9)

All my files end with a new line

The first line of all my files are exactly #!/usr/bin/env python3

My code should use the pycodestyle style (version 2.5.x)

The length of my files are tested using wc

All my modules have a documentation (python3 -c 'print(__import__("my_module").__doc__)')

All my functions have a documentation (python3 -c 'print(__import__("my_module").my_function.__doc__)')

All my functions and coroutines are type-annotated.
